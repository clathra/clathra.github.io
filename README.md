# clathra.com

Marketing website for Clathra, a marketing agency for the maritime and offshore industry.

Static site hosted on GitHub Pages. No build step: the files in this folder are the site root.

## Pages

- `index.html`: home (services, projects, contact)
- `privacy.html`: privacy policy
- `terms.html`: terms of use
- `accessibility.html`: accessibility statement (WCAG 2.1 AA)
- `404.html`: not found page

## Folders

- `images/`: logo, project images, favicons, share image (og-image)
- `assets/`: scripts (`consent.js`, the geo-aware cookie consent and analytics loader)

## Root config

- `sitemap.xml`, `robots.txt`, `llms.txt`: crawler, SEO and AI files
- `CNAME`: custom domain (clathra.com)

## Fonts

Loaded from Google Fonts: Bricolage Grotesque (headings), Hanken Grotesk (body), Space Mono (labels).
